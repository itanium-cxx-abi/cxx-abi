<div dir="ltr">The C++ standard requires that sizeof(nullptr) == sizeof(void*) (presumably to support programs that use sizeof(NULL) for some purpose) but does not appear to place any requirements on alignof(decltype(nullptr)). This clearly needs to be part of the ABI, and current implementations of the Itanium C++ ABI differ: Clang and EDG use alignof(void*). GCC uses 1.<div><br></div><div>We should pick a value and specify it in the ABI. alignof(void*) seems like a better answer to me. Thoughts?</div></div>
