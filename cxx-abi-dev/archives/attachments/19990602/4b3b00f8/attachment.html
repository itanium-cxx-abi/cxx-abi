<h4> Issue Status </h4>

In the following table,
the <b><i>class</i></b> column attempts to classify the issue on the
basis of what it likely affects.
The identifiers used are:
<dl>
<dt> call </dt>
<dd> Function call interface, i.e. call linkage </dd>

<dt> data </dt>
<dd> Data layout </dd>

<dt> lif </dt>
<dd> Library interface, i.e. API </dd>

<dt> g </dt>
<dd> Potential gABI impact </dd>

<dt> ps </dt>
<dd> Potential psABI impact </dd>

<dt> tools </dt>
<dd> May affect how program construction tools interact </dd>
</dl>
